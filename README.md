# Tower Defense Mini

Click or tap the map to place towers between waves and stop enemy waves before they reach the base.

## Upgraded Features

- Three tower roles: Pulse for fast cleanup, Beam for armored targets, and Frost for slowing enemy pushes.
- Mixed enemy waves with scouts, grunts, brutes, and shielded units.
- Upgradeable towers with visible range pulses, aim direction, shot traces, and impact particles.
- Build-time placement preview, wave threat counter, and clearer mobile-friendly controls.
